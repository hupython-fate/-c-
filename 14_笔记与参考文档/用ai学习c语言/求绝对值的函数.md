在 C 语言中，求绝对值的函数根据数据类型不同有多个，均定义在标准库中：

| 函数 | 原型 | 头文件 | 说明 |
|------|------|--------|------|
| `abs` | `int abs(int x);` | `<stdlib.h>` | 返回 `int` 型整数的绝对值 |
| `labs` | `long int labs(long int x);` | `<stdlib.h>` | 返回 `long` 型整数的绝对值 |
| `llabs` | `long long int llabs(long long int x);` | `<stdlib.h>` | 返回 `long long` 型整数的绝对值（C99 起） |
| `fabs` | `double fabs(double x);` | `<math.h>` | 返回 `double` 型浮点数的绝对值 |
| `fabsf` | `float fabsf(float x);` | `<math.h>` | 返回 `float` 型浮点数的绝对值（C99 起） |
| `fabsl` | `long double fabsl(long double x);` | `<math.h>` | 返回 `long double` 型浮点数的绝对值（C99 起） |

**示例：**
```c
#include <stdio.h>
#include <stdlib.h>  // 整数绝对值
#include <math.h>    // 浮点绝对值

int main() {
    int a = -5;
    double b = -3.14;

    printf("abs(%d) = %d\n", a, abs(a));
    printf("fabs(%.2f) = %.2f\n", b, fabs(b));
    return 0;
}
```

**注意：**  
- 在较旧的编译器中，`abs` 可能也定义在 `<math.h>`，但标准规定整数绝对值函数在 `<stdlib.h>`。  
- 如果使用 `abs` 处理浮点数，可能会被隐式转换为整数，导致结果错误，应使用 `fabs`。
