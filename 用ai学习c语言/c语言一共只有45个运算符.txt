# C语言所有运算符一览表

是的，C语言运算符确实不少。以下是**全部**运算符的分类列表：

## 1. **算术运算符**
- `+` 加法
- `-` 减法
- `*` 乘法
- `/` 除法
- `%` 取模（求余数）
- `++` 自增
- `--` 自减

## 2. **关系运算符**（比较）
- `==` 等于
- `!=` 不等于
- `>` 大于
- `<` 小于
- `>=` 大于等于
- `<=` 小于等于

## 3. **逻辑运算符**
- `&&` 逻辑与
- `||` 逻辑或
- `!` 逻辑非

## 4. **位运算符**
- `&` 位与
- `|` 位或
- `^` 位异或
- `~` 位取反（一元）
- `<<` 左移
- `>>` 右移

## 5. **赋值运算符**
- `=` 简单赋值
- `+=` 加后赋值
- `-=` 减后赋值
- `*=` 乘后赋值
- `/=` 除后赋值
- `%=` 取模后赋值
- `&=` 位与后赋值
- `|=` 位或后赋值
- `^=` 位异或后赋值
- `<<=` 左移后赋值
- `>>=` 右移后赋值

## 6. **条件运算符**（三元）
- `? :` 唯一的三元运算符

## 7. **逗号运算符**
- `,` 逗号

## 8. **指针运算符**
- `*` 解引用（间接访问）
- `&` 取地址

## 9. **结构体/联合运算符**
- `.` 成员访问
- `->` 指针成员访问

## 10. **其他运算符**
- `sizeof` 求大小
- `(type)` 强制类型转换
- `[]` 数组下标
- `()` 函数调用、类型转换

## 11. **C11新增**
- `_Alignof` 对齐查询

## 快速统计：
| 类别 | 数量 |
|------|------|
| 算术运算符 | 7个 |
| 关系运算符 | 6个 |
| 逻辑运算符 | 3个 |
| 位运算符 | 6个 |
| 赋值运算符 | 11个 |
| 其他运算符 | 10+个 |
| **总计** | **约45个** |

## 一个文件包含所有运算符示例：

```c
#include <stdio.h>

int main() {
    // 算术运算符
    int a = 10 + 5;  // +
    int b = 10 - 5;  // -
    int c = 10 * 5;  // *
    int d = 10 / 5;  // /
    int e = 10 % 3;  // %
    int f = 10; f++; // ++
    int g = 10; g--; // --
    
    // 关系运算符
    int r1 = (10 == 10); // ==
    int r2 = (10 != 5);  // !=
    int r3 = (10 > 5);   // >
    int r4 = (10 < 20);  // <
    int r5 = (10 >= 10); // >=
    int r6 = (10 <= 20); // <=
    
    // 逻辑运算符
    int l1 = (1 && 0);   // &&
    int l2 = (1 || 0);   // ||
    int l3 = !1;         // !
    
    // 位运算符
    int bit1 = 5 & 3;    // &
    int bit2 = 5 | 3;    // |
    int bit3 = 5 ^ 3;    // ^
    int bit4 = ~5;       // ~
    int bit5 = 5 << 1;   // <<
    int bit6 = 5 >> 1;   // >>
    
    // 赋值运算符
    int x = 10;          // =
    x += 5;              // +=
    x -= 3;              // -=
    x *= 2;              // *=
    x /= 4;              // /=
    x %= 3;              // %=
    x &= 1;              // &=
    x |= 2;              // |=
    x ^= 3;              // ^=
    x <<= 1;             // <<=
    x >>= 1;             // >>=
    
    // 条件运算符
    int y = (a > b) ? a : b;  // ? :
    
    // 逗号运算符
    int z = (a = 5, b = 10, a + b);  // ,
    
    // 指针运算符
    int num = 10;
    int *ptr = &num;    // &
    int value = *ptr;   // *
    
    // 结构体运算符
    struct Point { int x; int y; };
    struct Point p1 = {1, 2};
    struct Point *pPtr = &p1;
    int px = p1.x;      // .
    int py = pPtr->y;   // ->
    
    // 其他运算符
    size_t s = sizeof(int);  // sizeof
    int arr[5] = {0};
    int elem = arr[2];       // []
    // (type) - 强制类型转换
    float fval = (float)10;  // (type)
    
    return 0;
}
```

**其实不算少，共约45个运算符**。不过别担心，大多数都很直观，而且常用的只有20个左右。按分类学习，逐步掌握即可！
