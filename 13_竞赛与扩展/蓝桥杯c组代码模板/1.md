### 关于一次生成的数量建议
一次要 **3-4个模板最合适**：
1. 这些模板需要你动手敲、背下来，一次太多（超过5个）容易记混，当天消化不完；
2. 3-4个的量，你花1天把每个代码敲2-3遍就能完全记住，16个模板分5次就能过完，剩下的时间还能反复默写刷真题，节奏刚好。

### 1. 标准输入输出万能模板（所有题通用，防格式错）
```c
#include <stdio.h>   // 所有题必加的头文件
#include <string.h>  // 字符串/数组处理需要，直接加上不亏

int main() {
    // ✅ 全程只用scanf/printf，绝对不会超时、不会有格式错误
    // 示例：读入两个整数，输出它们的和，其他题改格式符就行
    int a, b;
    scanf("%d%d", &a, &b);  // 读多个变量，按顺序加&，空格分隔
    printf("%d\n", a + b);  // 输出最后必须加\n！否则会被判格式错误

    // 如果是读字符串：char str[1005]; scanf("%s", str); 
    // 如果是读浮点数：double x; scanf("%lf", &x); printf("%lf\n", x);

    return 0;
}
```
> 背诵点：所有题的开头都可以直接用这个框架，不用想头文件，输入输出就用这两个函数，C组永远不会出问题。

### 2. 一维数组遍历 + 求和/最值/统计模板
```c
#include <stdio.h>
#include <limits.h> // 用来初始化最值，可选，也可以自己设a[0]初始值

int main() {
    int n;
    scanf("%d", &n);       // 第一步：读数组长度
    int a[10005];          // C组开1e4大小足够，绝对不会超内存
    for (int i = 0; i < n; i++) {
        scanf("%d", &a[i]);// 遍历读入所有元素
    }

    // 👇 以下逻辑通用，直接套
    int sum = 0;                  // 1. 数组元素和
    int max_val = INT_MIN;        // 2. 最大值，初始化为最小整数
    int min_val = INT_MAX;        // 3. 最小值，初始化为最大整数
    int x, cnt = 0;               // 4. 统计x出现的次数
    scanf("%d", &x);              // 读要统计的目标数

    for (int i = 0; i < n; i++) {
        sum += a[i];
        if (a[i] > max_val) max_val = a[i];
        if (a[i] < min_val) min_val = a[i];
        if (a[i] == x) cnt++;
    }

    // 输出结果，按需改就行
    printf("sum=%d max=%d min=%d count of %d=%d\n", sum, max_val, min_val, x, cnt);
    return 0;
}
```
> 背诵点：一维数组的核心就是**单循环遍历**，所有的求和、最值、统计都是在遍历里顺便算的，不用多开循环，省时间。

### 3. 二维数组遍历 + 图形打印模板（C组高频题）
```c
#include <stdio.h>

int main() {
    // ========== 通用二维数组遍历 ==========
    int n, m;
    scanf("%d%d", &n, &m); // 读入：n行 m列
    int a[105][105];       // C组开100*100足够
    // 读入二维数组
    for (int i = 0; i < n; i++) {
        for (int j = 0; j < m; j++) {
            scanf("%d", &a[i][j]);
        }
    }
    // 按行输出二维数组
    for (int i = 0; i < n; i++) {
        for (int j = 0; j < m; j++) {
            printf("%d ", a[i][j]);
        }
        printf("\n"); // 每行结束必须换行！
    }

    // ========== 高频：金字塔图形打印（直接套） ==========
    int h;
    scanf("%d", &h); // 金字塔的高度
    for (int i = 1; i <= h; i++) {       // 遍历每一行
        // 1. 先打左边的空格：总高度 - 当前行号
        for (int j = 0; j < h - i; j++) {
            printf(" ");
        }
        // 2. 再打星号：当前行的星号数 = 2*行号 -1
        for (int j = 0; j < 2*i - 1; j++) {
            printf("*");
        }
        printf("\n"); // 换行
    }

    return 0;
}
```
> 背诵点：二维数组的核心是**双层for循环**，外层管行，内层管列；图形题的核心就是控制每行的空格和符号的数量，金字塔是最常考的，背会这个，其他菱形、方阵都能改出来。
