### 7. 选择排序完整模板【官方大纲明确考点：选择排序[3]】
```c
#include <stdio.h>
int main() {
    int n;
    scanf("%d", &n);
    int a[10005];
    for (int i = 0; i < n; i++) {
        scanf("%d", &a[i]);
    }

    // 👇 选择排序核心代码【大纲必考，必须背】
    for (int i = 0; i < n - 1; i++) { // 每轮确定i位置的最小数
        int min_idx = i; // 先假设当前i是最小数的索引
        // 找i后面所有数里，最小的那个的索引
        for (int j = i + 1; j < n; j++) {
            if (a[j] < a[min_idx]) {
                min_idx = j;
            }
        }
        // 把最小的数，交换到i的位置
        int tmp = a[i];
        a[i] = a[min_idx];
        a[min_idx] = tmp;
    }

    // 输出排序后的结果
    for (int i = 0; i < n; i++) {
        printf("%d ", a[i]);
    }
    printf("\n");
    return 0;
}
```
> 背诵点：选择排序的核心是**每轮选最小的放前面**，和冒泡的区别是，它每轮只交换一次，不用相邻挨个换。背会这个，C组要求的两个排序（冒泡+选择）就全齐了，排序题直接套。

### 8. 数位拆分模板（数字分解，初等数论基础）
```c
#include <stdio.h>
int main() {
    int num;
    scanf("%d", &num); // 输入要拆分的整数，比如1234

    int sum = 0; // 数位和（比如1234的数位和是1+2+3+4=10）
    int cnt = 0; // 数字的总位数

    // 👇 核心拆分逻辑，所有数字题通用
    while (num > 0) {
        int digit = num % 10; // 1. 取当前的个位（1234%10=4）
        printf("拆分出的当前位：%d\n", digit); 
        // 拿到digit后，你可以做任何处理：判断奇偶、统计、判断水仙花数...
        
        sum += digit; // 累加数位和
        cnt++;       // 位数+1
        num = num / 10; // 2. 删掉个位（1234/10=123）
    }

    printf("数位和：%d，总位数：%d\n", sum, cnt);
    return 0;
}
```
> 背诵点：C组所有数字类题的万能基础！比如判断回文数、水仙花数、数位求和、统计数字里的偶数个数，全靠这个。核心就两句：`%10取个位，/10删个位`，循环到num变成0，超级好背。

### 第二梯队1. GCD最大公约数 + LCM最小公倍数【官方初等数论核心】
```c
#include <stdio.h>

// 👇 辗转相除法求最大公约数，直接背这个函数
int gcd(int a, int b) {
    while (b != 0) {
        int tmp = b;
        b = a % b;
        a = tmp;
    }
    return a;
}

// 👇 最小公倍数，基于GCD计算
int lcm(int a, int b) {
    // 先除后乘，避免a*b直接溢出，C组也建议这么写
    return a / gcd(a, b) * b;
}

int main() {
    int a, b;
    scanf("%d%d", &a, &b);
    printf("最大公约数(GCD)：%d\n", gcd(a, b));
    printf("最小公倍数(LCM)：%d\n", lcm(a, b));
    return 0;
}
```
> 背诵点：初等数论的核心考点，C组的分数约分、周期问题、日期循环题，几乎都会用到这个。核心记住：`gcd(a,b) = gcd(b, a%b)`，直到b为0，a就是答案，这个函数直接背下来，遇到题直接调用就行。
